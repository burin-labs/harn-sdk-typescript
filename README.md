# harn-sdk-typescript

TypeScript SDK for the [Harn Agents API](https://github.com/burin-labs/harn-cloud/issues/72).

**Status:** bootstrap. The SDK is generated from the OpenAPI spec at [`burin-labs/harn`](https://github.com/burin-labs/harn) (epic [harn#631](https://github.com/burin-labs/harn/issues/631)). See the issue tracker on this repo for in-progress work.
